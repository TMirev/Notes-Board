# Notes-Board
Notes Board — HTML, CSS, JavaScript  
Built a responsive notes application that allows users to create, edit, and delete notes directly in the browser. 
Implemented persistent data storage using localStorage, ensuring notes remain saved after page refresh. 
Designed a fully responsive layout using modern CSS so the interface adapts smoothly across mobile, tablet, and desktop devices. 
Added interactive functionality with JavaScript, including dynamic note creation and real‑time content editing.
